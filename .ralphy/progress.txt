**What was built:**
ReportWizard.vue implements a step-by-step wizard for generating reports:
1. **Select Simulation** — Lists completed simulations from the Pinia simulation store (sessionRuns). Radio-style selection with scenario name, actions count, rounds, and relative timestamp.
2. **Report Type** — Three card options: Comprehensive Analysis, Executive Summary, Competitive Intelligence. Each with icon and description.
3. **Customize** — Optional custom instructions textarea, checkbox grid for section inclusion (engagement, sentiment, competitive, recommendations, agent profiles), and chart preference toggle (all/essential/none). Shows summary of all selections.
4. **Generate** — Calls POST /api/report/generate with customization payload. Progress bar with percentage, live section list as they're generated, shimmer loading states. Polls progress + sections endpoints every 3s. Error state with retry button.
5. **Review** — Chapter tab navigation, rendered markdown content, chapter prev/next navigation, export markdown button, link to full report view, "generate another" action.

**Additional features:**
- Stepper navigation bar with completed/active/pending states
- Back/Next footer navigation (disabled during generation/review)
- Report preferences saved to localStorage (`mirofish_report_wizard_prefs`)
- Uses existing composables: useSimulationStore, useToast
- Uses existing common components: AppButton, ShimmerCard
- Follows Intercom brand tokens (#2068FF blue, design token CSS vars)
- Cleanup on unmount (stops polling)
